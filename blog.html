Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents and manipulates HTML/XML documents as a structured tree.

How do you select an element from the DOM?

1.document.getElementById('id') – Selects an element by its ID.
2.document.querySelector('selector') – Selects the first matching element by CSS selector.
3.document.querySelectorAll('selector') – Selects all elements matching the CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element to manage events on multiple child elements. Instead of adding individual event listeners to each child, the parent listens for events and uses the event’s target property to handle specific child elements.

How do you manipulate an element's attributes and styles using the DOM?

1.setAttribute(name, value): Sets or updates an attribute.
2.getAttribute(name): Retrieves the value of an attribute.
3.removeAttribute(name): Removes an attribute.
